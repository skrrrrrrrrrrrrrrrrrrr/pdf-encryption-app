🚀 Getting Started

How do I merge PDF files?

To merge PDF files:

  • Click on the "Merge PDFs" tool
  • Upload at least 2 PDF files (up to 20MB each)
  • Drag and drop to reorder files if needed
  • Click "Merge PDFs" to combine them into one file
  • Download your merged PDF
What file formats are supported?

Currently, we only support PDF files (.pdf). All tools are designed specifically for PDF processing. Make sure your files are valid PDF documents.

Is there a file size limit?

Yes, each PDF file can be up to 20MB in size. This limit ensures fast processing and optimal performance. For larger files, consider compressing your PDF first.

🔒 Privacy & Security

Are my files stored on your servers?

No, absolutely not. All files are processed entirely in memory and are automatically discarded after processing. We never store your files or any user data on our servers.

How secure is the file processing?

Our platform uses enterprise-grade security:

  • All processing happens in secure, temporary memory allocation
  • Files are automatically purged after processing
  • No data is logged or tracked
  • Secure HTTPS connections for all uploads
Can I process confidential documents?

Yes, our privacy-first approach makes it safe to process confidential documents. Since files are never stored and processing happens in memory, your sensitive information remains secure.

🔧 Troubleshooting

Why is my file upload failing?

File upload issues can be caused by:

  • File size exceeding 20MB limit
  • Invalid or corrupted PDF file
  • Poor internet connection
  • Browser compatibility issues

Try reducing file size or using a different browser.

The processing is taking too long. What should I do?

Processing time depends on file size and complexity. Large files may take longer. If processing seems stuck:

  • Wait a bit longer (especially for large files)
  • Refresh the page and try again
  • Try with smaller files
  • Check your internet connection
I'm getting a "Backend Server Required" message

This message appears when the backend processing server is not running. This typically occurs in development environments. For the public website, this should not appear. If you're a developer, ensure both frontend and backend servers are running.

My encrypted PDF won't decrypt

Decryption issues are usually caused by:

  • Incorrect password entry
  • PDF using unsupported encryption methods
  • Corrupted or damaged PDF file

Double-check the password and ensure the PDF is not corrupted.
