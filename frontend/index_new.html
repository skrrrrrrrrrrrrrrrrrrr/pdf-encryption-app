Privacy & Security

🔒 Complete Privacy: Files are processed in memory and never stored on any server.

⚡ Local Processing: All operations happen on your device - no cloud uploads.

🛡️ Zero Data Collection: We don't track, store, or analyze your documents.

🗑️ Auto-Delete: Files are automatically discarded after processing.

Backend Server Required

The PDFMaster Pro backend server is not running.

To use PDF tools, you need to start the backend server:

cd backend && python main.py
