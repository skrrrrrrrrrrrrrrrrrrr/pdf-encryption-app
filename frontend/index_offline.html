Privacy & Security

🔒 Files are processed in memory and never stored.

All uploaded files are processed entirely in memory and are automatically discarded after processing.

No files or user data are stored on our servers.

All processing happens server-side with secure, temporary memory allocation.

Backend Server Required

The PDF Genie backend server is not running.

To use PDF tools, you need to start both the frontend and backend servers:

1. Start the backend server:
python -m uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
2. Start the frontend server:
cd frontend && python -m http.server 3000

See the README.md file for complete setup instructions.
